xlab Showcase 2022 - Interverse of Things

大爆走！オーディオレーシング

高 友康

Tomoyasu Taka

視覚情報を用いずに音声のみでプレイするオーディオゲームでは、音声のみで空間的な情報提示が困難であるといった課題があり、既存のレーシングゲームにおいては、プレイヤの回転が制限された簡易的なゲームシステムを用いたものが主流となっている。それに対し本研究では、プレイヤがより自由に空間的な移動・回転が可能なレーシング型オーディオゲームの制作し、プレイヤが自由に移動可能な状態でゲームを成立させるために、新たな誘導システムの設計・実装を行った。

Audio games, which are played using only sound without visual information, have issues of difficulty in presenting spatial information. Existing racing games are mostly based on a simple game system that restricts the player's rotation. We create a racing-type audio game that allows the player to move and rotate more freely in space by designing and implementing a a new guidance system which suppliment spatial information and makes the game work.
